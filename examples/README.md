# Examples

Examples in this directory must be redistributable.

- `demo_sic_mosfet`: synthetic 1200 V SiC MOSFET-like dataset for tests and tutorials.
- `minimal_device`: smallest built-in-emitter template with placeholder values.
- `plugin_hello`: minimal third-party plugin package showing entry point loading.

Do not add confidential datasheets or vendor files unless redistribution is explicitly allowed.
