# Support

Use GitHub Discussions for open-ended questions:

- which device family to support next,
- how to validate a model,
- how to interpret a datasheet curve,
- how to integrate a simulator,
- whether a contribution belongs in core or a plugin.

Use GitHub Issues for actionable work:

- reproducible extraction failures,
- broken generated netlists,
- simulator convergence failures,
- documentation mistakes,
- small feature requests with a clear outcome.

Please include public datasheet links instead of uploading third-party PDFs
unless redistribution is clearly allowed.
